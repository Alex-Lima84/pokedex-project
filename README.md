# Pokedex Project

The purpose of this project is to create a virtual Pokedex, in order to access several types of Pokemons and their stats.

This is a responsive project, made to be viewed in larger and small screens. 

<br/><br/>

## Applied Technology:

- HTML5;
- CSS3;
- JAVASCRIPT;

<br/><br/>
